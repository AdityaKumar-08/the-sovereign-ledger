# The Sovereign Ledger 🏛️

> India's Socio-Demographic & Resource Observatory — 2011–2051

A premium editorial data platform tracking India's population dynamics, resource sovereignty, and demographic transitions across 28 states and 8 union territories.

## 🎨 Design System

- **Typography**: Manrope (headlines) + Inter (body/labels)
- **No-Line Rule**: Section boundaries via tonal color shifts — never borders
- **Glassmorphism**: Navigation bar with backdrop blur
- **Color Palette**: Deep Indigo `#000666` · Sage Green `#1b6d24` · Terracotta `#3a0800`
- **No Sidebars**: Full-width editorial layouts on all pages

## 🚀 Getting Started

Simply open `index.html` in any modern browser. No build step required.

```bash
# Clone and open
git clone https://github.com/AdityaKumar-08/the-sovereign-ledger.git
cd the-sovereign-ledger
# Open index.html in your browser
```
